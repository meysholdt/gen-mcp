gen-mcp

OpenAPI spec in, MCP server repo out — fully automated.

1
▶

Trigger

A user starts the automation with two parameters: the target repo name and the OpenAPI spec URL.

repo_name SE-demo/my-mcp-server
openapi_url https://…/api.yaml
2
⚙

Generate

Ona creates a GitHub repo, generates the MCP server from the spec, builds it, and verifies it works. Progress is visible in real time.

3
📦

Receive

The user gets a ready-to-use Git repo with the MCP server code, a devcontainer, and a passing test — just clone and go.
